# Render gallery

Development screenshots captured while building SCUTTLE, in rough chronological
order. They were originally saved to the workspace root during early sessions
(when the working directory sat above this repo) and have been relocated here so
the game's visual history lives with the game.

Loose naming groups:

- **Concept / v1 passes** — `ocean-v*`, `ship-v*`, `sailing-v1`, `float-*`,
  `rig-v*`, `cutaway-v1`, `damage-v1`, `boarding-v1`, `broadside-v1`, `duel-v1`,
  `cannons-close`, `fp-v1`, `hud-v1`, `polish-v1`, `overhaul-*`, `sinking-stern`,
  `char-spike`.
- **Milestones** — `m7-*` … `m11-*` (see `../superpowers/plans/` for the matching
  milestone plans).
- **Round iterations** — `r9-*`, `r10-*` (the round-8/9/10 ocean + ballistics
  overhaul).

These are point-in-time captures, not runtime assets. Game textures live in
`../../public/assets/`.
