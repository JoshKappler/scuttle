# SCUTTLE audio — starter set + drop-in convention

The `.wav` files here are **original procedural placeholders** generated by
`scripts/gen-audio.mjs` (run `node scripts/gen-audio.mjs` to regenerate). They are
project-owned (effectively CC0) and exist so the audio engine works out of the box.
They are meant to be **replaced** with higher-fidelity audio later.

## How the engine finds sounds

`src/render/audio.ts` holds a `MANIFEST` mapping each sound **id** to a file path
under this folder. The id is what gameplay code plays (e.g. `audio.playAt("cannon", pos)`).
To upgrade a sound, do **either**:

1. **Same name:** drop your replacement in at the exact path below (keep the id's
   filename), or
2. **New file:** add the file and change that id's path in the `MANIFEST`.

Positional SFX (cannon, impact, crunch, sink, splash) **must be mono** or the 3D panner
won't spatialize them. Ambience and music may be stereo. `.ogg` is preferred for size;
the loader decodes whatever the browser supports (`.ogg`, `.wav`, `.mp3`).

## Licensing when you drop in premium audio (Steam endgame)

- **CC0** (Kenney, OpenGameArt CC0, Freesound CC0, BigSoundBank): no attribution needed,
  but list them in `../CREDITS.md` anyway.
- **Sonniss #GameAudioGDC bundle** (royalty-free, commercial-OK, **not** CC0): allowed for
  the marquee naval booms. Paste the bundle license into `LICENSE-sonniss.txt` and note
  which files came from it in `../CREDITS.md`. Do not resell the raw SFX.
