## Audio — real recordings (dropped in 2026-06-16)
Real downloaded audio now backs most of the soundscape. The originals were trimmed/
split/converted to OGG (music+ambience) / mono OGG (positional SFX) via ffmpeg; the
processed files live under `audio/sfx|ambient|music/` (see `audio/README.md`).

- **Wood-crack damage pool** — `audio/sfx/wood_crack_1..4.ogg`
  - Source: Freesound IDs **183450, 183451, 183452, 183453** by user **utsuru**
    ("wood crack" series) — https://freesound.org/people/utsuru/
  - ⚠️ **Confirm each sound's license on its Freesound page** (CC0 vs CC-BY). If CC-BY,
    attribution is required for the Steam build — keep this credit line.
- **Cannon** (`cannon_1..4.ogg`, split from one 4-shot take), **creak** (`creak.ogg`),
  **rope** (`rope_1..3.ogg`, sliced from one rigging-creak take), **coins**,
  **port_open** (ship's bell, 3rd strike trimmed), **ui_click**/**ui_buy**,
  **ocean_loop**, **wind_loop** (wind + creaking), **menu_theme**, **harbor**
  ("docked in town").
  - ⚠️ Sources were downloaded by the project owner; **fill in each file's library +
    license here before shipping** (Freesound/Pixabay/Sonniss/etc.). If any are CC-BY
    or Sonniss, note the author and paste the Sonniss license into `audio/LICENSE-sonniss.txt`.
