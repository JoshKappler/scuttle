# Quaternius / Kenney "Pirate Kit" (RAW DROP — gitignored)

- **Source:** https://quaternius.com/packs/piratekit.html
- **License:** CC0 (public domain — commercial OK)
- **Downloaded:** 2026-06-14 (glTF / FBX / OBJ / Blend).

This folder holds the **unzipped RAW pack** and is **gitignored** (large). Only the
optimized `web/` subset (committed) is what the game loads.

Low-poly pirate **props + environment** — barrels, crates, docks, palms, rocks,
cliffs, houses, skulls. (It also bundles blocky Kenney-style pirate **characters** —
Henry, Barbarossa, Anne, etc. — which we do **not** use; we have the Quaternius
Universal character.) We pull a few props from here for deck / scene dressing.

**To re-acquire:** download from the link above, unzip anywhere, re-copy the props
we use into `web/`.
